Abaqus无法启动的解决方法

2021-02-09 /
  • Abaqus
  • 笔记
 / 阅读耗时 1 分钟

今天打开Abaqus时，提示Licensing error:-15,10. Sysstem Error:10061 “WinSock: Connection refused”并无法启动，内心有些崩溃，不会又要重装软件了吧。

幸好卸载之前Google了一下，并成功找到解决方案。

  1. 用Everything找到Abaqus安装目录下的lmtools.exe文件，双击启动；
  2. 进入Start/Stop/Reread选项，点击start sever（我点击后无响应了一会，等待片刻即可）；
  3. 然后切换到Config Services选项卡，去掉Use Services方框的勾选，关闭lmtools.exe；
  4. 重新打开Aabaqus CAE，便可正常启动了。
